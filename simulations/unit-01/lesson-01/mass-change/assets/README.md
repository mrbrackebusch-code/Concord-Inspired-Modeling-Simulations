# Mass & Change Assets

This SVG set is for Lesson 1's macroscopic evidence suite.

- Use SVG for recognizable objects.
- Use Lab text boxes and simple shapes for readouts, fills, bubbles, and clouds.
- Keep Lesson 1 at the observable level. No particle visuals here.
- Record every imported or modified third-party asset in `ASSET_MANIFEST.md`.
- Prefer permissively licensed outside SVGs over weak custom placeholders when they materially improve quality.
- Do not import assets with unclear, `NC`, or `ND` terms.
